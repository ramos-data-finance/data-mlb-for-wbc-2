# Beginner's Guide to the WBC 2026 Prediction Pipeline

Welcome to the WBC 2026 Prediction Pipeline! This guide provides step-by-step instructions to help you utilize the prediction pipeline without any coding knowledge. 

## What is the WBC 2026 Prediction Pipeline?

The WBC 2026 Prediction Pipeline is a tool designed to forecast the performance of baseball players during the World Baseball Classic (WBC) 2026. This pipeline leverages historical data and machine learning models to generate predictions.

## Getting Started

### Step 1: Accessing the Repository

1. Open your web browser and go to the [GitHub repository](https://github.com/ramos-data-finance/data-mlb-for-wbc-2).
2. Make sure you are logged in to your GitHub account.

### Step 2: Downloading the Required Files

1. Navigate to the `releases` section of the repository.
2. Find the latest release and click on the assets to download the files you need.
3. Save the files to a directory on your computer where you can easily access them.

### Step 3: Setting Up Your Environment

1. Unzip the downloaded files to your chosen directory.
2. It's recommended to use an environment that supports Python. If you don't have Python installed, download it from the [official website](https://www.python.org/downloads/).
3. Once installed, open a command prompt or terminal.

### Step 4: Running the Prediction Pipeline

1. Navigate to the directory where you extracted the files using the command line. Use the `cd` command to change directories.
   
   Example:
   ```
   cd path/to/your/directory
   ```

2. Run the prediction script by typing the following command in the terminal:
   ```
   python predict.py
   ```

### Step 5: Viewing the Results

1. After the script executes, it will generate a results file. This file typically will be in the same directory.
2. Open the results file (often in CSV format) using software like Excel or Google Sheets to view the predictions.

### Step 6: Getting Help

If you encounter any issues or have questions, feel free to create an issue in the repository, and the community or maintainers can provide assistance.
